# CariCaj

CariCaj is a modern EV charging station discovery platform built for Malaysia.

The platform helps EV drivers quickly discover nearby charging stations, check real-time availability, receive community updates, and plan charging stops during their journeys.

---

# Vision

Become the most reliable EV charging discovery platform in Malaysia by combining official charging station information with real-time community contributions.

---

# Core Features

- Live GPS location
- Nearby charging stations
- Smart search
- Charging station details
- Community updates
- Photo uploads
- Fault reporting
- Route planning
- Favorite stations
- Google & Apple Sign-In
